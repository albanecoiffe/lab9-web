Advanced Web Programming - Vue Content Behaviour

  • Following the same idea and logic from our in-class example
    • https://github.com/anmarjarjees/vue.js-intro/blob/main/P1-Content-Behaviour/p1file4.html
  • Create a single page "index.html"
  • Add full CSS with RWD inside a css folder
  • Add another JS file inside a sub-folder named "data" to contain a variable/constant of type array that contains a list of objects
  • Each object in this file has at least 3 properties:
    • Title: any topic you prefer
    • Desc/Def: a short paragraph to talk about this title
    • img/images: an image link to any online free resource
  • Use the Vue CDN to connect to the Vue Framework library
  • Create a Vue app object with required properties according to our code examples
  • Specify the element inside your HTML page where you want to render the Vue properties plus the contents of the JS file inside the data folder:
    • Display "title" or "def" with the toggle click to switch between them
  • Adding two buttons to navigate through all the object elements inside the data folder
    • Button for next item
    • Button for the previous item
  • Upload your code on GitHub with a live demo link in the README.md file to run the live demo of your app in the browser.
